Wireframe

Wireframe é uma representação virtual de uma estrutura básica de uma página. Ele deve ser feito de maneira mais simples possível, mostrando apenas o essencial, como uma espécie de rascunho, sem cores ou imagens. Clique no botão ou escaneie o QrCode para obter o wireframe desta página.
